01

Colors

Our palette is built around a bold purple core with a dark navy anchor. Supplemental colors add energy and are used for data visualization, highlights, and illustrations.

Core colors

Purple 600 Primary Brand
Black Primary Text & BG

Supplemental colors

Purple 500 Hover / Active States
Blue 400 Links / Info
Cyan 300 Accent / Highlight
Yellow 300 Warning / Warmth

Usage guidelines

Do
  • Use Purple 600 as the primary CTA and brand color
  • Use Black for primary text and dark backgrounds
  • Use supplemental colors for data viz and illustrations
  • Maintain sufficient contrast (4.5:1 minimum for text)
Don't
  • Use supplemental colors as primary CTAs
  • Place Cyan or Yellow text on white backgrounds
  • Use more than 3 supplemental colors in one composition
  • Introduce off-palette colors without brand approval

04

Voice & Tone

OneSignal speaks like a sharp, technically-fluent teammate who genuinely wants to help you succeed. We're confident without being arrogant, and direct without being cold.

⚡

Sharp

We get to the point. No fluff, no filler. Every word earns its place. We trust our readers to be smart.

🔒

Dependable

We speak with confidence and back it up with data. Promises are kept. Commitments are real.

🙋

Helpful

We're geeky helpers — excited to solve problems. We explain the "why," not just the "how."

🎯

Clear

We avoid jargon unless our audience uses it first. Complex ideas are explained simply, not dumbed down.

🚀

Fast

Our language moves quickly. Short sentences. Active voice. We write the way our product delivers — instantly.

💬

Bold

We take a clear stance. We don't hedge or soften unnecessarily. We believe in what we're building.

In practice

✗ Not this
"We are pleased to offer a robust, enterprise-grade messaging solution that leverages cutting-edge technology to facilitate user engagement across multiple channels."
✓ This
"Send push notifications, emails, and SMS — all from one dashboard. Set it up in minutes."
✗ Not this
"Your message may have potentially experienced some delivery issues at this time."
✓ This
"3 messages failed to deliver. Check your API keys and retry."
✗ Not this
"Utilize our sophisticated segmentation capabilities to optimally target your audience demographics."
✓ This
"Segment by behavior, location, or custom data. Send the right message to the right person."

Word list

Use
  • Send / Deliver
  • Real-time
  • Segment
  • Users / Subscribers
  • Set up (not "onboard")
  • Dashboard
  • Notification
  • Scale
Avoid
  • Leverage (use "use")
  • Utilize (use "use")
  • Robust (use specific adjectives)
  • Synergy / Ecosystem
  • Journey (unless literal)
  • Revolutionize
  • Cutting-edge / State-of-the-art
  • Seamless (show it, don't say it)

Iconography

We use the Streamline Regular icon set — a clean, consistent line-style library that pairs well with Epilogue's sharp geometry. Icons communicate function, not decoration.

Library Streamline Regular
Style 1.5px stroke · Line / Outline
Sizes 16px · 20px · 24px · 32px · 48px
Format SVG preferred · PNG @2x fallback
Source streamlinehq.com ↗

Icon usage guidelines

Do
  • Use icons at their intended pixel sizes (16, 20, 24, 32, 48px)
  • Keep stroke weight consistent — always 1.5px in Streamline Regular
  • Color icons using brand palette tokens only
  • Pair icons with text labels in UI contexts
  • Use a single icon style throughout a product or document
Don't
  • Mix Streamline Regular with filled or other icon families
  • Scale icons to non-standard sizes (e.g. 18px, 22px)
  • Use icons without sufficient color contrast
  • Use icons alone as the only means to convey critical information
  • Add drop shadows or visual effects to icons
